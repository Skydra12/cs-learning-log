# C++ Basics

This repository contains my notes and example codes for learning the basics of C++ programming during military service.

## Purpose
- To understand C++ syntax and core concepts
- To compare C++ with C where necessary
- To practice object-oriented programming concepts

## Topics Covered
- Basic syntax and I/O (`cout`, `cin`)
- Control statements (if, loops)
- Functions and function overloading
- Pointers vs references
- Classes and objects
- Constructors and destructors
- Inheritance and polymorphism
- Basic STL usage (`vector`, `string`)

## Structure
Each topic is organized into a separate source file with clear comments and example outputs.

## Language
- C++ (ISO C++17)
